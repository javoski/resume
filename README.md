### A Simple Resume Template in HTML

#### Getting Started
- ##### Clone this repo
	```bash
	git clone https://github.com/javoski/resume
	cd resume
	yarn # or npm install
	```
- ##### Modify data.yml to your information
- ##### Run build
	```bash
	yarn run build # or npm run build
	```
- ##### Now you have your own resume in `dist` dir
